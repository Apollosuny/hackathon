# Face Recognition

## Setup environment

- Create new python environment
  ```bash
  python -m venv env
  ```
- Activate environment
  - Window:
  ```bash
  .\env\Scripts\activate
  ```
  - Ubuntu / Mac:
  ```bash
  source env/bin/activate
  ```
- Install packages
  ```bash
  pip install -r requirements.txt
  ```
- Setup Redis
  ```bash
  docker compose up
  ```
